What is Angular

  1. It is a front-end framework

  2. It teaches the browser new syntax to creating new HTML constructs what a web application needs - directives

Overview of concept of Angular

Template

HTML with additional markup, contains HTML, CSS and angular-specific elements or attributes

Directives

Extend HTML with custom attributes and elements

Model

The data that is show to the user and with which the user interacts

Scope

Context where the model is stored so that controllers, directives and expressions can access it. It connects controller and view

Expressions

Access variables and functions from the scope. Samiliar with javascript expression.

 

Compiler

Parses the template and instantiates directives and expressions

Filter

Formats the value of an expression for display to the user. Angular provides some filter, you also can create filter

View

What the user sees (the DOM)

Data Binding

Sync data between the model and the view

Controller

The business logic behind views

 

Dependency Injection

Creates and wires objects / functions

Injector

Dependency injection container

Module

Modules are used to organize the objects in an AngularJS application.Modules can either be used as as the ‘core’ of an application, and contain all of the classes used for it, or they can be used to group several objects that have similar functionality

Service

Reusable business logic independent of views

Why is Angular

MVC done right

Angular implements MVC by asking you to split your application MVC components, then just let Angular do the rest(by Scope)

A declarative user interface

Angular uses HTML to define the app's uer interface

Data models are POJO(Plain Old JavaScript Objects)

Scope is Angular's data models. All properties found on the scope object are automactically bound to the view by Angular

Behavior with directives

You can think directives are new HTML attributes, elements, it provides the function we need.

 

Flexibility with filters

Filters filter the data before they reach the view. For example: number, reversing the oder of an array. You also can create your own filter

DOM manipluations

Angular manipluations your HTML DOM you don't need to manipulation your DOM

Context aware communication

The PubSub system in Angular is precisely that. broadcast() will send a message to all children controllers, while emit() will send a message to all ancestors

Unit testing ready
